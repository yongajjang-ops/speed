# yonga's speed meter
# 2026.04.15
# 주행 중에 속도계 보려다 위험할 수 있어서, 10분마다 현재시간, 현재 속도,주행거리를 '음성' 들려주는 앱
● **PWA (Progressive Web App)**는 웹사이트를 마치 앱처럼 사용할 수 있게 해주는 기술입니다.

  쉽게 말하면

  일반 웹사이트인데, 스마트폰 홈 화면에 앱처럼 설치할 수 있는 것입니다.

  특징
  ┌─────────────────────┬───────────────────────────────┐
  │    일반 웹사이트    │              PWA              │
  ├─────────────────────┼───────────────────────────────┤
  │ 브라우저에서만 실행 │ 홈 화면에 아이콘 추가 가능    │
  ├─────────────────────┼───────────────────────────────┤
  │ 주소창 보임         │ 주소창 없이 앱처럼 보임       │
  ├─────────────────────┼───────────────────────────────┤
  │ 항상 인터넷 필요    │ 오프라인에서도 일부 작동 가능 │
  ├─────────────────────┼───────────────────────────────┤
  │ 앱스토어 불필요     │ 앱스토어 없이 바로 설치       │
  └─────────────────────┴───────────────────────────────┘
